# Portfolio

Personal portfolio website for Chanathip Aoibumrung (Tor) — Data & AI Engineer based in Bangkok, Thailand.

## Overview

A single-file static site (`index.html`) with no dependencies or build step. Open `index.html` in any browser to view.

## Features

- Responsive layout (max-width 800px)
- Automatic light/dark mode via `prefers-color-scheme`
- Sections: Experience, Skills, Education
